## OOP-Homework 1030

10 月 30 号理论课布置的作业，700 行低内聚高耦合的屎山（笑）

因为比较匆忙所以写成这样了，也没有写单元测试。再优化的话可能把一些该合并的逻辑合并一下，再仔细斟酌下几个方法的设计。

### How to run

把仓库拉到本地：

```bash
git clone git@github.com:Nikaidou-Shinku/OOP-Homework1030.git
cd OOP-Homework1030
```

构建并打包：

```bash
./gradlew jar
```

运行：

```bash
java -jar ./app/build/libs/app.jar
```
